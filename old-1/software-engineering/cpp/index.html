Call by Value vs. Call by Reference

A stack frame is created when a function is called. The stack frame contains information about the function such as the return address and all variables in the local scope.

  • Pass by Value

    The stack frame corresponding to the function call has a copy of the variable when a variable is passed by value into a function.

  • Pass by Reference

    When a variable is passed by reference the function has access to the actual data (at the original unchanged address) of the variable.

  • Pass a Pointer to a Variable by Value

    Passing a pointer as a parameter into a function creates a local pointer variable on the stack which allows you to access the original data that the pointer points to.
